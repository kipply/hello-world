# hello-world
A Node.js module created experimentally to print `Hello World` into the console.

## Installation
```sh
npm install hello-world --save
```

## Usage

### Javascript

```javascript
var helloWorld = require('hello-world');
helloWorld.helloWorld();
```

### TypeScript
```typescript
import { helloWorld } from 'hello-world';
helloWorld();
```
